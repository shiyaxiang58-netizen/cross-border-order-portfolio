# 作品集网页使用说明

这是一个不依赖数据库的静态网页，上传整个 `portfolio_site` 文件夹即可发布。

## 本地查看

直接双击 `index.html` 可以浏览页面并测试文件下载。

## 发布方式

- 国内访问优先：上传到支持静态网页托管的平台。
- 海外或技术岗位投递：可使用 GitHub Pages、Netlify 或 Cloudflare Pages。
- 发布时必须上传整个文件夹，保持 `assets`、`files`、`index.html` 和 `styles.css` 的相对位置不变。

发布后请用手机和无痕窗口测试：页面能否打开、PDF能否预览、Excel和Word能否下载。
